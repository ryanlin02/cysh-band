# 網站聯絡與帳號維護指引

本文件記錄嘉義高中管樂隊暨校友管樂團網站的公開聯絡入口與共同帳號原則。目標是讓網站、社群與資料保存不綁在單一個人的私人帳號上。

## 1. 目前公開聯絡入口

- 網站頁面：`contact.html`（「聯絡與資料更正」）。
- 對外按鈕使用：`contact@cysh.band`。
- Cloudflare Email Routing 只將此公開地址轉寄至非公開的私人收件信箱；該目的地不可寫入網站或 repo。
- 網頁不直接顯示私人收件信箱；寄件者點選「寄送電子郵件」才會開啟自己的郵件軟體。
- 適用事項：內容勘誤、校友資料補充、歷史素材、照片／文字授權、要求更正或下架、網站技術意見。

處理來信時，不要把寄件者的電話、地址、身分證字號、私人聯絡方式、未公開名冊或群組截圖直接貼回網站資料夾。只有確認可公開的資料，才可依既有公開資料流程整理。

## 2. Cloudflare 收件轉寄管理

公開地址：`contact@cysh.band`

用途：網站聯絡收件、內容更正與素材權利相關來信。Cloudflare Email Routing 已設定唯一規則：寄至此地址的信轉寄至一個非公開目的地。

調整收件人時：登入 Cloudflare，選擇 `cysh.band` → **Email Routing** → **Destination addresses**，新增並驗證新的目的地；再到 **Routing rules** 將 `contact` 規則改為新目的地。完成後才移除舊目的地。

注意事項：

1. 網站與 repo 只記錄公開地址，絕不記錄私人目的地、密碼或驗證碼。
2. DNS 的 Cloudflare MX 與 SPF 紀錄是收件轉寄所需設定，不要任意刪除。
3. 此功能目前只處理收信；若直接從私人 Gmail 回覆，對方仍可能看見私人 Gmail 地址。
4. 未來需要以 `@cysh.band` 名義寄信、多位管理者共用信箱或登入其他官方服務時，再評估 Google Workspace 或其他寄信服務；目前不需要付費方案。
