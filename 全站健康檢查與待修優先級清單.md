  - 先補重要人物、聯演核心人物、具公開資料者。
  - 不要為了補數量而建立資料不足的個人頁。

## E. 暫不處理但要知道

### E1. `tidy` HTML 檢查工具不適合直接作為判斷依據

- 本機 `tidy` 版本較舊，不認得 `nav`、`header`、`main`、`section`、`figure` 等 HTML5 標籤。
- 因此它輸出的多數錯誤不是網站實際問題。
- 後續若要做 HTML 標準檢查，應改用較新的 HTML validator 或以瀏覽器測試為主。

### E2. 目前未做 Playwright 視覺截圖測試

- 原因：本機環境未安裝 Playwright。
- 本次已用本機 HTTP、靜態連結、資料檔、SEO 欄位與 CSS 規則做替代檢查。
